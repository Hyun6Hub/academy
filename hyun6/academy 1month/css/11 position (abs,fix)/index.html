postion (abs,fix)

  • postion(abs,fix)속성을 이용하면 독립적인 위치 설정이 가능
  • postion(abs,fix)속성이 적용되었을 경우
    • maring의 auto값이 적용되지 않는다
    • block레벨의 전체넓이가 사라진다
    • inline레벨도 크기가 적용된다
    • postion(abs,fix)속성이 적용된 요소는 부모요소가 인식하지 못한다
    • postion(abs,fix,rel,sticky)모두 z-index값을 가지게 된다
  • 좌표는 (left,right)중하나 (top,bottom)중 하나를 사용한다
    • position(abs)좌표 기준은 position속성을 가지고 있는 부모요소
    • position(fix)좌표 기준은 브라우저 기준
