html기초개념

html문서의 개념

  • html : Hyper Text Marteup Language, DOM을 설계하는 언어
  • DOM : html문서 구조
  • Element : 웹문서를 이루는 나의 요소
  • Element 는 태그(tag)와 속성(attribute)으로이루어진다
  • Element 는 node를 형성한다(자손,자식,부모,형제)
    • 자손(descendent)
    • 자식(children)
    • 부모(parent)
    • 형제(siblings)
  • 웹표준 : W3C에서 제시하는 DOM을 구축하는 문법을 준수해야
  • 웹접근성 : 누구나 어디서나 사용이 가능하도록 설계하기를 향
  • 크로스브라우징 : 모든 브라우저나 기기에서 호환되는 웹이나 앱을 계해야 함
  • 시멘틱언어 : 브라우저가 이해할 수 있는 언어로 웹문서를 작성하길 권장

sementic element

  • heading요소: 제목, h1~h6
  • p: 관련된 문장의 범위
  • blockquote: 인용(오늘날씨가 매우 추워요), cite 출처(네이버날씨)
  • em, strong: 중요, b: 시각적으로 강조 중요하거만 뽑아옴 오늘 날씨추워요
  • smail: 보조적인 의미 오늘ms(micro soft)주식상승 00영화(00감독)
  • time: 일자나 시간을 표시
  • addres: 주소,전화번호,업체명
  • span,div: 기능이나 의미가 없음 디자인목적으로 레이아웃
