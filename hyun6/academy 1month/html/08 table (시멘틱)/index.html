시멘틱 테이블

  • th요소는 데이터의 분류명을 의미, scope속성을 이용해서 스크린리더가 데이터를 읽어주는 방향을 결정
  • scope 속성
    • scope="col" (하단 데이터를 읽기 전에 상단의 th를 읽는다)
    • scope="row" (오른쪽 데이터를 읽기 전에 왼쪽의 th를 읽는다)
    • scope="colgroup" (하단 th를 읽기 전에 상단의 th를 읽는다)
    • scope="rowgroup" (오른쪽 th를 읽기 전에 왼쪽의 th를 읽는다)
  • tbody요소는 데이터의 범위 (반드시 포함해야 함)
  • thead는 th요소의 범위
  • tfoot은 데이터와 관련 없는 범위
