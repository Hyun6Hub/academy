form

  • form 엘리먼트
    • 기능: 데이터를 전송하는 기능, 전송할 데이터의 범위
    • action 속성: 데이터를 전송받아서 처리하는 주소
    • method 속성 :
      get (쿼리스트링을 이용해서 데이터를 전송)
      post (보안이필요하거나, 바이너리 파일을 전송)
  • fieldset : 폼요소의 유사데이터 범위
  • legend : fieldset의 제목
  • label : 입력양식에 대한 참조설명, title속성으로 대체해도 무방
  • select 메뉴에서 required속성을 사용하기 위해서는 첫번째 option요소에 value값을 지정하지 말것
  • option요소는 label속성을 제공한다
