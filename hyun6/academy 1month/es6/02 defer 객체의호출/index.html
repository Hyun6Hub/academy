엘리먼트 호출

  • getElement
    • document.getElementByTagName('태그이름'): 배열을 반환
    • document.getElementByClassName('클래스이름'): 배열을 반환
    • document.getElementById('아이디이름'): 해당 아이디 요소 반환
  • query
    • document.querySelector('css 선택자'): 첫번째 요소 한개를 반환
    • document.querySelectorAll('css 선택자'): 모든 요소를 배열로 반환
