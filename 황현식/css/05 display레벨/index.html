display레벨과 정렬

display레벨에 의해서 화면 출력방식과 문법작성의 유효성이 결정된다

display레벨의 종류

block

  • 부모의 전체 넓이를 가지고 출력이된다
  • 한줄을 통쨰로 차지한다
  • 크기속성을 가질 수 있다
  • h1~h6,p,div,article, form, li, section, main 등

inline

  • 줄내림이 되지 않는 요소들
  • 크기속성 가질 수 없다
  • a, span, b ,em, smaill, i

inline-block

  • 줄내림이 되지 않는 요소들
  • 크기를 가질 수 있음
  • img, video, svg, canvas, input, button, textarea, select, ifreame

display레벨과 정렬

  • 글자를 정렬할 떄는 글자의 부모요소에서 text-align속성을 이용한다
  • line-heihgt : 줄간격을 제어하는 속성, 요소의 높이와 동일하게 설정하면 수직중앙정렬 사용 할 수 있음
  • vertical-align: 글자들간의 상중하 위치를 정렬
  • block레벨의 수평정렬시 margin의 auto값을 적용한다 !주의 : margin의 auto값은 block레벨만 사용 가능
