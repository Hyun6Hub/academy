css기초

Cacsading Style Sheet

css의 특징

  • 웹문서를 시각적으로 디자인하느 목적의 언어
  • 우선순위가 존재하는 디자인 양식
  • 선택자{프로퍼티:값}

선택자(selector)

타입선택자

  • 문법: 태그네임{프로퍼티:값;프로퍼티2:값}
  • 자식선택자가 상위우선순위를 가진다
  • 선택자별로 다양한 디자인 적용이 불가능하다

class선택자(클래스선택자)

  • 문법: .클래스이름{프로퍼티:값}
  • 타입선택자보다 상위우선순위를 가진다
  • 선택자별로 다양한 디자인 적용이 불가능하다
  • 중복호출이 가능(통일성을 유지한 디자인설계와, 유지보수가 용이함)

id선택자

  • 문법: #아이디이름{프로퍼티:값}
  • 클래스선택자보다 상위우선순위를 가진다
  • 중복호출이 불가능(css설계목적이 아닌 html문법을 준수하기 위해 사용)

인라인스타일

  • 엘리먼트의 스타일 속성을 추가후 프로퍼티와 값을 추가
  • 스크립트를 이용해서 css를 동적으로 제어하는데 사용됨
  • 최상위 우선순위를 가진다
