flex-direction

  • flex-direction:row (기본값)
  • 자식요소들이 옆으로 배치된다
  • 자식들이 부모요소의 높이와 동일하게 늘어남 (stretch)
  • flex-direction:coulmn
  • 자식요소들이 상하로 배치된다
  • 자식들이 부모요소의 높이와 동일하게 늘어남 (stretch)
  • flex-direction:row-reverse
    자식요소들을 오른쪽 부터 배치
  • flex-direction:column-reverse
    자식요소들을 아래쪽 부터 배치

    align-items:start

  • 자식요소들의 stretch속성이 없어진다
    • flex-direction row일 경우 상단축
    • flex-direction coulmn일 경우 왼쪽축

    align-item:center

    • flex-direction row일 경우 수평중앙축
    • flex-direction coulmn일 경우 수직중앙축

    align-items:end

    • flex-direction row일 경우 하단축
    • flex-direction coulmn일 경우 오른쪽축

    justiy-content

    • 자식요소의 위치를 설정
    • justiy-content : start (기본값)
      • flex-direction row 왼쪽
      • flex-direction coulmn 왼쪽
    • justiy-content : center
      • flex-direction row 수평중앙
      • flex-direction coulmn 수직중앙
    • justiy-content : end
      • flex-direction row 오른쪽
      • flex-direction coulmn 아래쪽
    • justiy-content : space between
      • flex-direction row 좌우분할
      • flex-direction coulmn 상하분할
