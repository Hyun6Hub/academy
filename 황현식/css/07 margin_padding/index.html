margin과 padding

  • margin(간격)은 요소의 바깥쪽 거리
    • margin: 10px(상하좌우 전부)
    • margin: 0 10px 20px 30px (위 오른쪽 아래 왼쪽)
    • margin: 0 auto (상하 좌우)
    • margin-left, margin-top, margin-bottom, margin-right,
    • 첫번째 블록요소는 margin-top이 적용되지 않는다(부모요소에 border나 padding이 적용되지 않았을 경우만)
    • 첫번째 블록요소는 margin-bottom이 적용되지 않는다(부모요소에 border나 padding이 적용되지 않았을 경우만)
    • box-sizing:border-box (padding)과border크기를 요소 안쪽으로 적용한다
    • inline요소는 상하 마진과 상하패딩이 적용되지 않는다
    • block요소간의 상하 마진은 큰값하나만 적용된다
    • padding과 amrgin을 %단위로 적용할 경우 부모요소의 width 기준으로 적용됨
  • padding(여백)은 요소의 안쪽 공간
